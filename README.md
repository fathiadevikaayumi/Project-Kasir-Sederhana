# Project-Kasir-Sederhana
Aplikasi kasir sederhana berbasis Python menggunakan Tkinter. Program ini memiliki fitur login dengan batas percobaan, form input data pembeli dan barang, perhitungan total harga, pembayaran, serta menampilkan struk transaksi.
Fitur
Login dengan batas maksimal percobaan
Input nama pembeli
Input nama barang
Input jumlah produk
Input harga produk
Perhitungan total harga otomatis
Menghitung kembalian
Menampilkan struk pembayaran

Teknologi yang Digunakan
Python
Tkinter (GUI Library)
